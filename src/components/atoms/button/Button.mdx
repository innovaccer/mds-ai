# Triggers

Triggers are used to invoke AI. Common UI elements such as buttons, inputs, chips, etc. can act as triggers to invoke AI.

## Types

Triggers come in different types based on the existing UI elements -

### AI Buttons

Buttons can be used to trigger AI actions. These buttons could be Primary, Secondary or Icon buttons.

#### Primary Button

Primary button can be used to trigger primary AI actions, hence drawing attention to the primary action of the page. An additional magic sparkle icon is present to indicate the presence of AI.

<Canvas of={ButtonStories.Primary} />

#### Basic button

Basic button can be used to trigger secondary AI actions. An additional magic sparkle icon is present to indicate the presence of AI.
